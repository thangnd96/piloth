# Example — Task Routing

Task: “Sửa bug validation khi submit form.”

```text
Rules: coding behavior
Knowledge: form specification
Runtime: task lifecycle
Skills: testing/debug workflow nếu có
Agents: software engineer
Tools: test runner
Evaluation: correctness + regression gates
Governance: không cần approval nếu thay đổi nhỏ, reversible
```

Không cần nạp toàn bộ Memory, Tools index hoặc các domain không liên quan.
