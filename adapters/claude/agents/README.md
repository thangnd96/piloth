# Claude Native Subagents

No generic subagents are enabled by default.

PilothOS defaults to single-agent execution. If a task needs team behavior, use `pilothOS/agent-teams/` first. Create native Claude subagents only after a real repeated need is validated and logged.

Piloth Team role contracts live in `pilothOS/agents/team-roles/`. Native Claude subagents should be created only when a validated task proves the adapter needs real subagent files.
